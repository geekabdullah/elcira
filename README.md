# Circuit Express - Tech E-Commerce Platform

⚡ A modern, lightweight e-commerce platform for tech products with admin dashboard and email notifications.

## Features

- 🛍️ **Product Catalog** - Browse electronic components, digital products, and learning materials
- 👥 **Agent Directory** - Connect with technical support agents
- 🔧 **Admin Dashboard** - Manage products and agents with password protection
- 📧 **Email Notifications** - Get instant alerts for new orders via Gmail SMTP
- 💾 **File-based Storage** - Simple JSON files for easy data management

## Tech Stack

- **Backend**: Node.js + Express
- **Frontend**: HTML, CSS, JavaScript
- **Email**: Nodemailer with Gmail SMTP
- **Storage**: JSON files
- **Deployment**: Ready for Render, Heroku, or any Node.js host
